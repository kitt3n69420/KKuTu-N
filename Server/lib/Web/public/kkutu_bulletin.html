  • 모바일에서 십자말풀이의 보드 길이가 항상 정사각형으로 표시되지 않는 문제

  • 솎솎 모드에서 큰 보드 사용 시 한 줄에 13문자만 표시되는 문제
  • 솎솎 모드에서 점수판이 내려가는 문제
  • 그림퀴즈 CSS에서 요소가 드래그되는 문제
  • 규칙 설명이 올바른 위치에 표시되지 않는 문제
  • 방장이 연습 중, 다른 플레이어가 모두 준비를 마쳤을 때 방장이 경고 없이 연습이 끝나지만 자동으로 게임이 시작이 안되는 문제
  • 연습 중에 잠수 방 폭파 경고가 표시되는 문제
  • 작은 모드 레이아웃을 계산할 때 관전자까지 고려하는 문제
  • 작은 모드 레이아웃에서 점수가 중앙정렬되지 않는 문제
  • 연습모드에서 본게임에 갑자기 참여되면 두 게임에 모두 들어가 있고 점수가 NaN이 되는 문제
  • 서로게이트(4바이트 유니코드 문자ㅡ 보통 이모지 등)가 각각 1글자로 취급되는 문제
  • 로비의 유저 목록에 표시되는 유저 수가 실제보다 많아 보이는 문제
